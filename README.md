# Task Management System

This project was developed as part of the Project Management course.

## Project Description
A simple task management system that allows users to add and delete daily tasks.

## Methodology
This project follows Agile project management principles.

### Scrum
- The project was divided into 2 sprints
- Each sprint lasted one week

### Kanban
- A Kanban board was used to track tasks
- Tasks were moved between To Do, In Progress, and Done columns

## Tools
- GitHub

## Features
- Add tasks
- Delete tasks

## Project Duration
2 weeks
